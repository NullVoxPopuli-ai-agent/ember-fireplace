# ember-fireplace

> gather at the fireplace, have a chat over _glimmer_ and _ember_

A what if...? Simulation!

- What if we move all resources that are available for ember in one place?
- What if we use resources that are not yet standardized (through) RFC, but hit more of the current times?
- What if the resource here is helpful to human _and_ machine?
- What if we continue these resources, we feed the machines (by accident)?

This all happens, when we meet at the fireplace (hence the name).
Obviously, this is a community research project, with the aim to push this upstream into ember itself.

Here you go:

```sh
pnpm install
pnpm dev
```
